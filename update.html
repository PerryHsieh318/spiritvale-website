SpiritVale 更新公告

這裡整理 SV LAB 每一次版本更新內容， 包含新增功能、系統優化與問題修正， 方便玩家快速查看最新版本資訊。

LATEST UPDATE

Version 2026.07.22.5

2026 / 07 / 22
LATEST

✨ 本次更新重點

  • 全新物品管理系統
  • 召喚師 Buff 優化
  • 自動重連全面升級
  • 新增 AFK 保護功能
  • 介面操作改善
  • 多項問題修正

📦 全新物品管理系統

本次全面升級物品列表介面， 讓搜尋、設定與地圖傳送操作更加方便。

  • 全面升級物品列表介面。
  • 可直接搜尋任何物品。
  • 可為指定物品建立專屬設定。
  • 已建立自訂規則的物品，系統將不再自動販售。
  • 支援一鍵傳送至該物品的掉落地圖。
  • 傳送介面新增中文地圖名稱。
  • 每張地圖新增建議等級資訊。

⚔️ 召喚師優化

  • 優化召喚師 Buff 判定與施放邏輯。
  • 提升召喚師整體戰鬥表現與穩定性。

🔄 自動重連全面升級

自動重連流程更加完整， 降低斷線、讀取異常或角色卡住造成的中斷。

  • 偵測讀取畫面異常時，可自動關閉並重新開啟遊戲。
  • 登入角色後會自動重新啟動程式。
  • 提升自動重連成功率。
  • 修正角色可能卡在讀取畫面的問題。
  • 改善自動選擇伺服器與角色的流程。
  • 強制重連流程更加穩定。
  • 第一次重連失敗時，系統會自動再次嘗試。

🛡️ 新增 AFK 保護功能

安全設定新增 Reconnect if no kills。

可設定角色幾分鐘內沒有擊殺怪物時， 自動重新啟動遊戲、重新登入並繼續執行。

可避免角色因異常停留原地， 卻沒有被即時發現的情況。

🎨 介面優化

  • 點擊搜尋框後，搜尋文字會立即清除。
  • 改善搜尋框的可見度與操作方式。
  • 減少小幅度的介面卡頓。
  • 提升整體 HUD 的操作流暢度與穩定性。

🔧 問題修正

  • 修正自動登入時可能選錯伺服器的問題。
  • 修正連線尚未開始，卻被誤判已完成重連的情況。
  • 改善斷線及讀取畫面中斷後的恢復能力。

更新提醒

建議所有使用者更新至 2026.07.22.5 版本，以取得最新功能與穩定性改善。
